## Super-Market Software
SuperManager is a <b>Java Swing desktop application</b> developed as <i>a university group project</i> for the <ins>Software Engineering course</ins>.

The application supports supermarket operations through <b><ins>a role-based access system</ins></b>, where each user is <ins>granted permissions and functionalities</ins> according to <i><ins>their responsibilities within the organization</ins>.</i>
